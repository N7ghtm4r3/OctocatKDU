Parameters

frequencyVisibility

The temporal frequency to display or not the dialog (also when there is an update available)

locale

The locale language to use

notShowAtNextLaunchOptionEnabled

: whether enable the option for the user to not display, so not be warned about a new update, at the next launches

appName

The name of the application where the dialog will be shown

currentVersion

The current version of the application, it will be used to check if is currently the latest version available

onUpdateAvailable

The action to execute if there is an update available and the dialog is shown

dismissAction

The action to execute when the dismiss button has been clicked

confirmAction

The action to execute when the confirm button has been clicked
